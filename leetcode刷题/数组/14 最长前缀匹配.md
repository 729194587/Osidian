编写一个函数来查找字符串数组中的最长公共前缀。
如果不存在公共前缀，返回空字符串 `""

好像只能遍历。
初版代码：
```c
#include<string.h>
#include<stdio.h>
#include<stdlib.h>
char* longestCommonPrefix(char** strs, int strsSize) {
    int i=0;
    int minlen=sizeof(strs[0])/sizeof(char);
    for(i=0;i<strsSize;i++)
    {
        int len=sizeof(strs[i])/sizeof(char);
        minlen=minlen<len? minlen:len;
    }
    char *result=(char *)malloc(sizeof(char)*minlen+1);//末尾的\0
    int j=0;
    while(j<minlen)
    {
        char c=strs[0][j];
        int flag=1;
        for(i=0;i<strsSize;i++)
        {
            if(strs[i][j]!=c)
            flag=0;
        }
        if(!flag)
        break;
        else
        result[j]=c;
        j++;
    }
    return result;
}
```

优化后：
```c
char* longestCommonPrefix(char** strs, int strsSize) {
        if(strsSize == 0) return ""; //越界判断
    int i=0;
    int minlen=strlen(strs[0]);//求长度
    for(i=0;i<strsSize;i++)
    {
        int len=strlen(strs[i]);
        minlen=minlen<len? minlen:len;
    }
    char *result=(char *)malloc(sizeof(char)*minlen+1);//末尾的\0
    int j=0;
    while(j<minlen)
    {
        char c=strs[0][j];//注意c的索引
        int flag=1;
        for(i=0;i<strsSize;i++)
        {
            if(strs[i][j]!=c)
            {
            flag=0;
            break;
            }
        }
        if(!flag)
        break;
        else
        result[j]=c;
        j++;
    }
    result[j]='\0';
    return result;
}
```
唉，改了好多次代码。

strs是char\*\*类型，因为字符串本身是一个字符指针，指向字符串的空间，而字符串数组又指向多个字符串的空间。
然后就是修改对长度的计算。
`int len=strlen(strs[i]);`

此外，字符串需要添加末尾的'\0'。

C++版的：
```cpp
#include <string>
#include <vector>
#include <algorithm> // std::min

std::string longestCommonPrefix(const std::vector<std::string>& strs) {
    if (strs.empty()) return ""; // 空数组处理

    // 先找出最短字符串长度
    size_t minlen = strs[0].size();
    for (const auto& s : strs)
        minlen = std::min(minlen, s.size());

    std::string result;
    for (size_t j = 0; j < minlen; ++j) {
        char c = strs[0][j];
        bool allMatch = true;
        for (size_t i = 1; i < strs.size(); ++i) {
            if (strs[i][j] != c) {
                allMatch = false;
                break;
            }
        }
        if (!allMatch) break;
        result += c; // 逐字符加入结果
    }

    return result;
}
```
`std::vector<std::string>`表示标准库容器vector，用于存储字符串。
`std` 是 **standard** 的缩写，表示 C++ 标准库。
`::` 是作用域解析运算符，用来访问命名空间或者类里的成员。

如果写 `vector<int> v;` 而不加 `std::`，编译器找不到 `vector`，因为它在 `std` 里。
但是
```c
using namespace std;
vector<int> v;  // 就可以省略 std::
```
不过，**一般不推荐在大型项目里直接用 `using namespace std;`，会增加命名冲突风险。**
`&` 表示**引用（reference）**，类似于传指针。
strs还有empty函数和size函数。
